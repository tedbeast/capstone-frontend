Please run these commands in your terminal, one at a time, to implement... 

1. npm install 
2. npm install react-scripts 
3. npm install react-router 
4. npm install --save @fortawesome/react-fontawesome @fortawesome/fontawesome-svg-core @fortawesome/free-solid-svg-icons 5. npm start 

References: url for fontawesome (#4) icons: https://fontawesome.com/icons
